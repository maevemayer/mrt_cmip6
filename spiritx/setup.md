#### Installation

##### create environment
in the spirit-x shell run:
`conda create -n 01lagranto -c conda-forge ipykernel python=3.9.9`
reopen session and run:
`conda activate 01lagranto`
`python -m ipykernel install --user --name my-kernel --display-name="01lagranto"`
install dependencies as 
`conda install --file conda_environment.txt`
Now you are setup for this module
